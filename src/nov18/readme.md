## HashSet vs LinkedHashSet vs TreeSet ? 

1. HashMap 
HashMap is a collection which store data in form of key- value pair 
where keys must be unique and values could be duplicate

keys are in random order 
1 null key is allowed 


2. LinkedHashMap
collection which is in form of key value pair. 
it maintains insertion order 
keys must be unique 
can contain 1 null key



TreeMap
collection which is in form of key value pair. 
natural sorted order 
it does not allow any null keys
